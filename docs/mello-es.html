Mello

Descripción del Proyecto

¿Qué es?

Mello es una aplicación web responsiva de vista única construida con Vue.js. Inspirada en Trello, permite a los usuarios crear, gestionar y priorizar sus listas de tareas. Además, los usuarios pueden personalizar la interfaz seleccionando entre una variedad de imágenes de fondo predefinidas.

¿Cuál era el objetivo?

El objetivo fue desarrollar una aplicación web responsiva utilizando el framework Vue.js y el sistema Material Design como parte del Bootcamp Fullstack de Desarrollo Web de CareerFoundry. Este proyecto buscaba mostrar habilidades tanto de front-end como de diseño, cumpliendo con las mejores prácticas modernas de desarrollo web.

¿Qué herramientas se utilizaron?

Cliente: Vue, Material Design
Entorno de Desarrollo: Node.js
Alojamiento: GitHub Pages

Construcción de la App

Requisitos técnicos para crear la app:

  • La app debe construirse usando Vue.js.
  • La interfaz debe diseñarse con Material Design a través de Vue Material.
  • La app debe ser responsiva y mostrarse bien en todos los tamaños de pantalla.
  • La app debe ser serverless.
  • La app debe permitir al usuario personalizar la interfaz.
  • La app debe desplegarse en GitHub Pages.

El código fuente está alojado en GitHub, una plataforma colaborativa que soporta control de versiones y gestión de proyectos. Puedes acceder al código aquí.

El usuario puede crear nuevas listas y tareas, y luego arrastrarlas para cambiar el orden de las listas y priorizarlas. Al hacer clic en una tarea, se muestran detalles específicos que permiten editarla.

El usuario puede elegir entre una selección de imágenes de fondo para personalizar la interfaz.

La aplicación es totalmente responsiva, asegurando una experiencia fluida en dispositivos móviles, laptops y escritorios.

Desafíos Enfrentados

El mayor desafío en este proyecto fue aprender los fundamentos del framework Vue.js, especialmente después de haber trabajado bastante con React en proyectos anteriores. Tuve que “cambiar de marcha,” lo que me brindó una experiencia práctica invaluable con diferentes frameworks. Además, este proyecto me dio la oportunidad de trabajar con el sistema Material Design por primera vez, ya que anteriormente sólo había trabajado con Bootstrap en otros proyectos.
