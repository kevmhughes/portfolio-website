Flowerheads

Descripción del Proyecto

¿Qué es?

Flowerheads es una aplicación e-commerce moderna y totalmente responsiva construida con React, React Context API, Next.js, Stripe y Sanity. Ofrece una experiencia de compra completa, con navegación de productos, carrito fluido e integración completa con Stripe para el procesamiento de pagos.

¿Cuál era el objetivo?

Construir y desplegar una aplicación e-commerce full-stack completamente responsiva con funcionalidad de pagos, un diseño moderno, animaciones, la capacidad de agregar y editar productos fácilmente mediante un CMS sin cabeza, y una interfaz de pago completamente integrada como Stripe.

¿Qué herramientas se utilizaron?

Cliente: React
Servidor: Next.js
Procesamiento de Pagos: Stripe
CMS sin cabeza: Sanity
Entorno de Desarrollo: Node.js
Alojamiento: Vercel

Características Clave

Los usuarios pueden:

  • Explorar los productos.
  • Añadir productos al carrito.
  • Eliminar productos del carrito.
  • Ver el carrito.
  • Pagar de forma segura con Stripe.

Revisión de los productos en el carrito.

Pago seguro mediante la interfaz de procesamiento de pagos completamente integrada de Stripe.

Más Funcionalidades

Los propietarios de la app pueden:

  • Agregar fácilmente productos a la base de datos mediante Sanity Studio.
  • Agregar detalles del producto, imágenes y precios a la base de datos mediante Sanity Studio.
  • Modificar detalles del producto fácilmente usando Sanity Studio.
  • Eliminar productos de la base de datos fácilmente con Sanity Studio.
  • Actualizar rápidamente imágenes principales y banners en la página de inicio usando Sanity Studio.

El propietario de la aplicación puede agregar y modificar fácilmente los productos a través del CMS sin cabeza, Sanity Studio. Esto permite actualizaciones y gestión de productos en tiempo real.
