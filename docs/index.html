My Projects

LiveDocs

LiveDocs is a fully responsive online word processor and collaborative document app built with React, Next.js, TypeScript, and Liveblocks. Similar to Google Docs, it offers a seamless writing experience, empowering users to work together on documents in real time. Multiple individuals can edit, comment, and track changes simultaneously, making collaboration effortless and efficient.

Flower Heads

Flower Heads is a fully responsive and modern full-stack e-commerce application built with React, Next.js, Stripe, and Sanity. It provides a seamless shopping experience, from browsing products to secure payment processing. See case study here.

Mello

Mello is a fully responsive, single-view Kanban to-do list app built with Vue.js, offering a streamlined and intuitive way to organize tasks and manage workflow. Discover more about the development journey in the full case study here.

ChitChat

ChitChat is a native mobile application built using React Native, offering a messaging service similar to WhatsApp. With features like real-time communication and media sharing, it provides users with a seamless messaging experience. See the full case study here.

Krypt

Krypt is a Web 3.0 blockchain app for sending and receiving cryptocurrency, built with React, Solidity, Tailwind, and Vite. It enables users to securely manage transactions via a MetaMask account by connecting their digital wallet.

Yariga

Yariga is a full-stack real estate management dashboard for desktops, built with the MERN stack. It offers powerful, intuitive tools for managing properties and agents, tracking reviews, and handling transactions, all within a sleek, user-friendly interface.
